# 🎮 Flutter 2048 Game

A modern implementation of the classic **2048 puzzle game** built with **Flutter** and **Dart**. Players swipe tiles on a 4×4 grid to merge matching numbers, increase their score, and aim to reach the **2048 tile**. The project demonstrates game logic implementation, algorithm design, gesture handling, and clean Flutter architecture.

---

## ✨ Features

* 🎮 Classic 2048 Gameplay
* 👆 Swipe Gesture Controls
* 🔢 Tile Merging Algorithm
* 💯 Dynamic Score Tracking

## 📂 Project Structure

```text
lib/
│
├── core/
├── models/
├── services/
├── screens/
├── widgets/
├── utils/
└── main.dart
```

### Folder Description

* **core/** – Game constants and core business logic.
* **models/** – Tile and game state models.
* **services/** – Handles tile movement, merge logic, score calculation, and game-over detection.
* **screens/** – Game screens and user interface.
* **widgets/** – Reusable widgets such as the game board, tiles, score card, and restart button.
* **utils/** – Helper methods and utility functions.
* **main.dart** – Entry point of the application.

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/kiranmai456/flutter-2048-game.git
```

### Navigate to the project

```bash
cd flutter-2048-game
```

### Install dependencies

```bash
flutter pub get
```

### Run the application

```bash
flutter run
```
